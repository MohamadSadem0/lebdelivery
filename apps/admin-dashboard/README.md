# Admin Dashboard

React/Vite admin dashboard for platform approvals, monitoring, receipts, and operational controls.

## Run

```bash
cp .env.example .env
npm install
npm run dev
```

Default URL:

```text
http://localhost:5174
```

The dashboard expects an admin account and uses `VITE_API_BASE_URL`, defaulting to:

```text
http://localhost:8080/api/v1
```
