# Delivery Web

React/Vite delivery portal for the Lebanon Local Commerce & Delivery Platform.

This is a web companion to the driver mobile app. It is useful for desktop dispatch visibility and driver web testing, but the production driver experience should remain mobile-first.

## Run Locally

```bash
cp .env.example .env
npm install
npm run delivery-web
```

Default URL:

```text
http://localhost:5176
```
