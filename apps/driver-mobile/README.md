# Driver Mobile

Expo React Native foundation for independent and store-owned drivers.

## Run

```bash
npm install
npm start
```

Set the API URL with:

```bash
EXPO_PUBLIC_API_BASE_URL=http://localhost:8080/api
```

## Current Scope

This app includes providers, navigation, secure token storage helper, location helper, API client placeholder, auth Zustand store, and simple placeholder screens. Driver assignment, active delivery workflow, earnings, and cash collection logic are intentionally not implemented yet.
