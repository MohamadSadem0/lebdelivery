# Client Mobile

Expo React Native foundation for clients ordering products, requesting services, tracking orders, viewing receipts, and managing profile/support.

## Run

```bash
npm install
npm start
```

Set the API URL with:

```bash
EXPO_PUBLIC_API_BASE_URL=http://localhost:8080/api
```

## Current Scope

This app includes providers, navigation, secure token storage helper, API client placeholder, auth Zustand store, and simple placeholder screens. Final UI and business workflows are intentionally not implemented yet.
